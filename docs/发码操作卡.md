# 发码操作卡（¥99 / 码 · 人工收款人工发码）

## 流程

1. 用户在 https://bd.cxodex.com/register 看到指引：扫收款码付 ¥99 → 加微信发付款截图。
2. **你确认微信到账后**，在本机 PowerShell / Git Bash 跑下面这条命令（10 秒）：

```bash
ssh -i ~/.ssh/cxodex_hk ubuntu@43.160.214.195 "cd ~/bd-app && export \$(grep DATABASE_URL .env.local) && node scripts/seed-invite.mjs '' 1"
```

输出形如 `invite code: BD-3F7A2C9E (max_uses=1)` —— 把这个码发给用户即可。

- **一码一用**（max_uses=1）：注册成功即失效，转发无效。
- 想自定义码面：把 `''` 换成你想要的码，如 `node scripts/seed-invite.mjs VIP-ZHANG 1`。
- 想给某人多个名额：把 `1` 改成名额数。

## 查询 / 对账

```bash
# 看所有码的使用情况
ssh -i ~/.ssh/cxodex_hk ubuntu@43.160.214.195 "cd ~/bd-app && export \$(grep DATABASE_URL .env.local) && node -e \"const{neon}=require('@neondatabase/serverless');neon(process.env.DATABASE_URL)('SELECT code, used_count, max_uses, created_at::date FROM invite_codes ORDER BY created_at DESC').then(r=>console.table(r))\""

# 看注册用户（谁用了哪个码）
ssh -i ~/.ssh/cxodex_hk ubuntu@43.160.214.195 "cd ~/bd-app && export \$(grep DATABASE_URL .env.local) && node -e \"const{neon}=require('@neondatabase/serverless');neon(process.env.DATABASE_URL)('SELECT email, invite_code_used, created_at::date FROM users ORDER BY id DESC LIMIT 20').then(r=>console.table(r))\""
```

## 码的现状（2026-07-02 起）

- `BD-LAUNCH`（原 20 次通用码）：**已停用**（max_uses=0）。
- `DEV-TEST`（原本地开发码）：**已删除**，替换为随机开发码（见 docs/deploy.md 私密备注），仅本地冒烟用。
- 正式发放：全部走上面的一次性码命令。

## 边界情况

- 用户付了钱码丢了/没用就丢了：对账查 `used_count=0` 的码可直接重发同一个码。
- 用户要退款：微信原路退回即可；若已注册，删用户 `DELETE FROM users WHERE email='...'`（级联删其数据）。
